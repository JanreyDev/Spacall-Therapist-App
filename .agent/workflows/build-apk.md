---
description: How to build a release APK for the therapist application
---

1. Open a terminal in the therapist app directory: `c:\Users\AMD\Workspace\spacall_therapist_app`
2. Run the following command:
// turbo
```powershell
flutter build apk --release
```
3. Once finished, the APK file will be located at:
`build\app\outputs\flutter-apk\app-release.apk`
